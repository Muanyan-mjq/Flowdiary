# 随心耶 v1.0.0 — 第一个正式版本 🎉

> 记录生活，随心而动 —— 一只萨摩耶陪你写日记

这是随心耶的第一个正式版本。从零开始，一个人、一台电脑，完完整整做出了自己人生中第一个 App。

---

## 能做什么

**写日记** — 四步向导（天气→心情→事件→写作），支持 Markdown、语音输入、配图，离开自动保存草稿

**看回忆** — 回忆隧道随机推送历史日记，时间线按月份浏览，心情日历可视化情绪轨迹

**玩塔罗** — 78 张塔罗牌 + 翻牌动画 + 12 星座详解 + 八字运势

**养萨摩耶** — 写日记升级，Lv1 萌新出窝 → Lv6 耶中之王，写越多等级越高

**日签墙** — 每日签到收集好句子，6 色渐变卡片，可收藏、生成二维码分享

**保护隐私** — 伪装密码（两个密码进不同空间）+ 指纹/面容锁 + 本地加密

---

## 技术

Flutter 3.x 开发，同时支持 Android / iOS。本地数据 JSON + SecureStorage，云端基于 Supabase（可选）。完整项目结构、注释清晰，适合 Flutter 初学者参考。
