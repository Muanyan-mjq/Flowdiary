# 随心耶 App 完整介绍

## 基本信息
- **名称**：随心耶
- **类型**：日记 / 生活记录 / 心情陪伴
- **平台**：iOS + Android（Flutter 开发）
- **核心形象**：萨摩耶小狗（白色萨摩耶，温暖治愈）
- **Slogan**：记录生活，随心而动
- **目标用户**：年轻女性为主，喜欢记录生活、治愈系内容、轻度玄学

---

## 整体设计风格
- 暖白底色 + 圆角卡片 + 柔和阴影
- 主题色可自定义（默认天蓝 #87CEEB）
- 大量使用萨摩耶小狗插画形象
- 中文界面，温暖口语化文案
- 细腻动画（缩放、淡入、滑动、翻牌）

---

## 五大主 tab

### 1. 日记（首页）
- **卡片流首页**：PageView 横向滑动，三张卡片——新笔记、最近日记、默认卡片
- **默认卡片**：萨摩耶递信大图 + 黑色条幅写「感谢相遇」「一份萨摩小耶的来信」，点击进入给用户的第一封信
- **写日记向导**：四步流程——选天气 → 选心情 → 选事件标签 → 写作
  - 写作页支持 Markdown 实时预览（标题、加粗、斜体、引用、待办、列表）
  - 语音输入（中文语音识别）
  - 添加配图
  - 6种背景色切换（默认/羊皮纸/浅灰/暗黑/护眼/樱花）
  - 字号调节（5档）
  - 撤销/重做
  - 离开自动保存草稿
- **动态文案系统**：根据天气+时间段自动变化小狗形象和文案（深夜/清晨/早上/上午/中午/下午/傍晚/晚上各有不同小狗图片）
- **日期选择器**：点击顶部「日记本 ▼」弹出日历，跳转任意日期的日记

### 2. 发现（回忆隧道）
- **数据面板**：累计日记数、记录天数、总字数
- **随机回忆**：随机推送一篇历史日记，每天不同
- **时间线浏览**：按年月分组，滑动查看所有日记
- 顶部标题「MEMORY TUNNEL」+ 呼吸脉冲动画

### 3. 社区（专注/待办）
- **待办清单**：添加每日任务，支持打卡完成
- **番茄钟**：专注计时器，圆形进度条动画
- **数据统计**：完成率、专注时长统计
- 顶部根据时间显示「早安/上午好/下午好/晚上好 + 时间」
- 完成激励文案（「全部完成，今天超棒」「已完成的 X/Y，继续加油」）

### 4. 心情（塔罗 + 星座 + 八字）
- **塔罗牌**：完整 78 张塔罗（22张大阿尔卡纳 + 56张小阿尔卡纳），点击翻牌动画，显示牌名、emoji、含义解读
- **星座**：12 星座详解——守护星、元素、性格关键词、优缺点、幸运色/数字、合拍星座、深度解读文案
- **八字**：输入出生信息，生成八字简析
- 顶部三个 tab 切换（塔罗/星座/八字）

### 5. 我的（个人主页）
- **头像 + 昵称 + 签名**：可编辑，头像支持自定义上传
- **等级系统**（萨摩耶养成）：
  - Lv1 萌新出窝（0篇）→ Lv2 拆家小将（5篇）→ Lv3 汪汪达人（20篇）→ Lv4 忠诚伙伴（50篇）→ Lv5 天使耶（100篇）→ Lv6 耶中之王（200篇）
  - 每个等级有专属配色和边框
- **数据统计**：日记数、天数、字数
- **快捷入口**：心情日历、日签墙、草稿箱、云端备份
- **菜单**：主题色、我的收藏、意见反馈、关于
- **个人空间**：独立页面，自定义背景色（6种），头像大图，最近日记

## 辅助功能

### 日签墙
- 6色柔美渐变卡片网格
- 每日签到收集好句子
- 可收藏、分享（生成二维码）
- 全屏详情页 + 分享图生成

### 心情日历
- 月历视图，每天按心情着色（开心=金、难过=蓝、平静=绿、惊喜=粉、生气=红）
- 对应 emoji 图标
- 点击日期查看当天日记

### 月度视图
- 卡片滑动选择月份
- 入场动画从1月滑到当前月
- 年份快速切换 overlay
- 每张月卡显示日记数量

### 侧边栏
- 用户信息卡片 + 头像
- 问候语（时间自适应）
- 连续记录天数（火焰图标）
- 菜单：月度视图、列表视图、收藏、草稿(红点)、云端备份(状态指示灯)

### 安全隐私
- **伪装密码**：设置两个密码，一个进真实日记，一个进假日记空间
- **应用锁**：生物识别（指纹/面容）+ 密码
- 本地数据加密存储（FlutterSecureStorage）

### 云端
- 基于 Supabase
- 云端同步日记、待办
- 云端认证（注册/登录）

### 其他
- 首页标题日期选择器（日历跳转任意日期日记）
- 全局搜索日记
- 日记 Markdown 渲染（标题/粗体/斜体/引用/待办/列表/分割线）
- 日记配图画廊（横向滚动+点击放大）
- 收藏功能
- 草稿箱（红点提醒）
- 意见反馈
- 天气集成

## 设计风格要求
- 暖白底色，圆角卡片，柔软阴影
- 萨摩耶形象（白色狗狗）贯穿所有物料
- 字体：中文用圆体/无衬线，温暖亲切
- 配色：天蓝主题色 + 暖色点缀
- 风格：治愈、温暖、可爱但不幼稚
